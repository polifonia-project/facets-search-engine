See Help to find example of supported metadata files.
They should be in json format, strictly named after the document id of the corresponding already indexed document. If you are not sure about the document id, or which index the document belongs to, please see Dashboard.
Once metadata is uploaded, it can not be erased. If you wish to update any information, you may upload a new metadata file to overwrite the previous version.
